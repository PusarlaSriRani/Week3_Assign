# 🛒 E-Commerce Management System – SQL Worksheet

## 📌 Overview

This repository contains my **Week 3 SQL Practice** based on an **E-Commerce Management System**. It covers database schema creation, sample data insertion, relationships, joins, subqueries, window functions, CTEs, transactions, indexing, and dashboard queries using PostgreSQL.

The project is designed to strengthen SQL concepts through practical business scenarios.
